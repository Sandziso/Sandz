Project Overview

Eswatini Down Syndrome Association (EDSA) Website is a comprehensive digital platform designed to support families, raise awareness, and facilitate donations for Down Syndrome initiatives across Eswatini. The website features modern design with Eswatini-specific elements and mobile money integration tailored for the local market.

Database Architecture

  • Password hashing with bcrypt algorithm (cost 12)

  • Session management with automatic timeout and regeneration

  • Transaction status tracking and automated notifications
