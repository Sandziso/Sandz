Project Overview

HomewareOnTap is a complete e-commerce solution built from scratch to transform a social media-based homeware business into a professional online retail operation. The platform features a comprehensive admin dashboard, customer portal, secure payment integration, and automated inventory management.

Database Architecture

Designed and implemented a comprehensive MySQL database with 15+ tables including:

  • Users: Multi-role authentication system (customer, admin, manager)
  • Products: Complete inventory management with stock alerts
  • Orders: Full order lifecycle tracking and payment processing
  • Reviews: Customer rating and review system with moderation
