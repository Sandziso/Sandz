Project Overview

Mamba Rentals Eswatini is a sophisticated rental property platform designed to modernize the property rental market in Eswatini. The platform connects landlords with prospective tenants, streamlines the booking process, and provides comprehensive property management tools.

Technical Implementation

Database Architecture

Designed and implemented a comprehensive MySQL database with 15+ interconnected tables:

  • Users: Multi-role authentication system with profile management
  • Properties: Complete property listings with media and amenities
  • Bookings: Full booking lifecycle tracking with status management
  • Payments: Secure transaction processing and financial records
  • Reviews: Rating and feedback system with moderation
  • Messages: Real-time communication logs between users

Security Implementation

  • Password hashing with bcrypt algorithm (cost 12)
  • CSRF token protection for all forms and actions
  • SQL injection prevention using prepared statements
  • XSS protection through comprehensive input sanitization
  • Secure file upload validation with type and size restrictions
  • Session management with automatic regeneration

Payment System

Integrated multiple payment methods tailored for the Eswatini market:

  • Credit/Debit card processing via PayFast gateway
  • Secure bank transfer integration
  • Mobile money support (MTN Mobile Money, Eswatini Mobile)
  • Automated transaction logging and receipt generation
  • Payment status tracking and notification system
